# Fullstack Lab

This repository contains multiple **independent full-stack projects** built with different technologies.

Each project is self-contained and may include its own backend, frontend, and optional Docker setup.

## Purpose

This repository is created for:

- Learning and practicing full-stack development
- Experimenting with backend & frontend integrations
- Applying clean architecture principles
- Building a portfolio of practical, real-world applications

## Tech Stack (varies per project)

**Backend**
- Python
- FastAPI
- Uvicorn

**Frontend**
- HTML, CSS, JavaScript

**Containerization**
- Docker
- Docker Compose

**Version Control**
- Git & GitHub

## Projects

###  climate-sim-app
A simple climate simulation application.

- Backend-driven
- Uses mathematical models to simulate climate-related values
- Includes a minimal frontend for interaction

###  weather-api-app *(coming / in progress)*
A weather application consuming real-world weather APIs.

- Frontend-focused
- External API integration
- Async JavaScript & modular architecture

> Each project contains its own README with setup and usage instructions.
