
This repository contains multiple full-stack projects built with different technologies.
Each project includes its own backend, frontend, and optional Docker setup.

This repository is created for:

Learning full-stack development

Experimenting with backend & frontend integrations

Practicing clean architecture

Building a portfolio of practical applications
_____________________________________________________________________

Backend: FastAPI, Python, Uvicorn

Frontend: HTML, CSS, JavaScript (or React in future projects)

Containerization: Docker & Docker Compose

Version Control: Git & GitHub

______________________________________________________________________
Enter the project folder:

Run Backend
cd backend
uvicorn app.main:app --reload

Run Frontend:
cd frontend
python -m http.server 5500

Or run everything with Docker:
docker compose up --build
